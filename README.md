# Armario 

Este es un servicio web que permite alojar tus prendas y tus outfits de manera local. Instalas la pagina web a través de tu navegador y lo ejecutas como una app. 

Todos los datos se manejan de manera local, no se envían a ningún servidor, simplemente son almacenados en tu navegador. En cualquier momento, se pueden exportar o importar. Si se usan navegadores no usuales, como Brave u otros más privados, es probable que los datos se eliminen, por lo que sea necesario exportar los datos una vez se hayan generado, y guardarlos de manera segura, para cuando vuelva a abrirse sean importados correctamente.
